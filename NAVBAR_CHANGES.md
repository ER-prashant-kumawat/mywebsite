## How to Use

### Click Admin (Not Logged In)
1. Click "Admin" button in navbar
2. Go to login page
3. Enter password
4. See admin dashboard

### Click Logout (Logged In)
1. Click "Logout" button in navbar
2. Return to home page
3. Admin session cleared
4. Button shows "Admin" again
